# Vehicle Tracking System

This project is a web-based vehicle tracking system developed using ASP.NET and MSSQL.  
It is designed to help vehicle service businesses manage customers, vehicles, services, spare parts, and payments in a structured way.

## Features
- User authentication system
- Vehicle tracking by license plate
- Customer information management
- Service and maintenance history tracking
- Spare parts stock management
- Service fee and payment tracking

## Technologies Used
- ASP.NET
- C#
- Microsoft SQL Server (MSSQL)
- HTML
- CSS

## Project Purpose
This project was developed to gain hands-on experience in backend development, database design, and real-world business logic implementation.

## Notes
This is a personal learning project.  
Sensitive information such as database credentials has been removed for security reasons.
